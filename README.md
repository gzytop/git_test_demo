# QR Forge

一个零构建、零后端的二维码生成器网页小工具 Demo。

## 功能

- 输入文本或链接生成二维码
- 支持下载二维码 PNG 图片
- 提供示例内容快速体验
- 响应式布局，支持深色模式

## 在线 Demo

已启用 GitHub Pages：

https://gzytop.github.io/git_test_demo/

## 本地运行

直接用浏览器打开 `index.html` 即可。

> 本项目使用 CDN 加载 `qrcode` 浏览器库，首次生成二维码需要网络可访问 jsDelivr。

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- qrcode.js
